## Ключевые сущности (физические)

### ESP32

Микроконтроллер, подключённый через USB-Serial (UART, 115200 бод). Центральный узел связи между ПО и физическими компонентами:

- Принимает JSON-команды, возвращает JSON-ответы
- Управляет бюреткой через двухфазные (fill, empty, rinse, dose) и однофазные (stop) JSON-команды
- Каждые ~300 мс шлёт broadcast-пакет с состоянием датчиков, объёмом бюретки (volume_ml), скоростью (speed_ml_min) и статусом двигателя
- Управляет шаговым двигателем, клапаном, считывает ADC (pH/ОВП-электрод) и DS18B20 (температура)

### Дозирующий модуль: бюретка и клапан

- **Номинальный объём** — максимальный объём жидкости, который вмещает бюретка (мл)
- **Шаговый двигатель** — перемещает поршень. Позиция измеряется в шагах от нижнего концевика
- **Нижний концевик (limit switch FULL)** — датчик положения поршня внизу (бюретка заполнена)
- **Верхний концевик (limit switch EMPTY)** — датчик положения поршня вверху (бюретка пустая)
- **Клапан** — переключает поток жидкости: `input` (забор из бутыли с титрантом) / `output` (выдача в титровальный сосуд)
- **Остаточный объём (leftover volume)** — текущий объём жидкости в бюретке (мл)

### Датчики

- **pH/ОВП-электрод** — измеряет pH (ОВП) через электрический потенциал (мВ). Показания зависят от температуры
- **Датчик температуры (DS18B20)** — измеряет °C, используется для температурной коррекции pH

## Физические процессы (бизнес-логика)

### Дозирование

Дозирование выполняется на ESP32. Tauri отправляет команду `burette.doseVolume` с параметрами volume_ml и speed_ml_min через двухфазный протокол. ESP32 управляет клапаном, двигателем и концевиками. Результат приходит после завершения.

### Титрование (линейное)

Цикл на каждую порцию:

1. **Дозировать** порцию титранта (шаг, например 0.1 мл)
2. **Ожидать стабилизации** pH/мВ:
   - Измерение каждые 100 мс
   - Дрейф (мВ/мин) по скользящему окну (10 измерений)
   - Стабилизация: дрейф < порога (20 мВ/мин) или таймаут (100 с)
3. **Зарегистрировать точку** (объём, pH, производная)
4. **Рассчитать производную** dpH/dV (фильтр Савицкого–Голея, окно 3, полином 3)
5. **Детектировать точку эквивалентности** при превышении порога производной (интерполяция между последними двумя точками)
6. **Критерии остановки:**
   - Точка эквивалентности найдена + добавлен постаналитический объём (2 мл)
   - Достигнут максимальный объём титранта
   - Превышено максимальное время анализа
   - Пользователь нажал «Стоп»
7. **Сохранить результаты** (кривая, точки эквивалентности) в CSV

### Калибровка pH

Измерить mV в буферах (pH 4.01, 7.01, 10.01) при известной температуре → рассчитать наклон (slope) и смещение (offset) по формуле Нернста: `E = E0 + slope * (pH - 7)`, где `slope(T) = slope_25C * (T + 273.15) / 298.15`

### Инициализация бюретки (homing)

Инициализация выполняется на ESP32 после подачи питания (stepper_home). Tauri не управляет инициализацией бюретки.

## Глоссарий

| Термин                                        | Определение                                                                                                                                                                                                                                                                                           |
| --------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Бюретка**                                   | Устройство для дозирования титранта, представляющее собой стеклянный цилиндр с одним отверстием для прохода жидкости, поршнем, обеспечивающим перекачку жидкости, приводимым в поступательное движение шаговым двигателем по одной оси, и двумя концевиками, ограничивающими крайние положения поршня |
| **Концевик (limit switch, limit sensor)**     | Датчик положения поршня бюретки (нижний (MIN - бюретка заполнена) / верхний (MAX) - бюретка пустая)                                                                                                                                                                                                   |
| **Титровальный сосуд**                        | Ёмкость, в которой титрант смешивается с аналитом в процессе титрования; в ней также находится электрод и датчик температуры                                                                                                                                                                          |
| **Бутыль с титрантом**                        | Ёмкость, в которой находится основной запас титранта для заполнения бюретки по мере необходимости                                                                                                                                                                                                     |
| **Клапан**                                    | Устройство для переключения направления подачи титранта ("input": из бутыли с титрантом в бюретку, "output": из бюретки в титровальный сосуд)                                                                                                                                                         |
| **Потенциал (мВ)**                            | Электрический сигнал pH/ОВП-электрода                                                                                                                                                                                                                                                                 |
| **Титрант**                                   | Раствор с известной концентрацией, добавляемый из бюретки                                                                                                                                                                                                                                             |
| **Титрование**                                | Процесс добавления титранта для определения концентрации аналита                                                                                                                                                                                                                                      |
| **Метод титрования**                          | Совокупность настроек процесса титрования (скорость дозирования, порог стабилизации (мВ/мин), порог производной, максимальный объём титранта, время анализа и прочее)                                                                                                                                 |
| **Кривая (график) титрования**                | График, отражающий изменение pH или потенциала электрода в зависимости от объема добавленного титранта в процессе титрования                                                                                                                                                                          |
| **Точка эквивалентности (Equivalence Point)** | Объём титранта, при котором количество реагента эквивалентно количеству аналита                                                                                                                                                                                                                       |
| **Конечная точка титрования (End Point)**     | Объём титранта, при котором достигается заданное значение pH или потенциала электрода                                                                                                                                                                                                                 |
| **Broadcast**                                 | Периодическое сообщение ESP32 (~300 мс) с текущими показаниями                                                                                                                                                                                                                                        |
| **АЦП (ADC)**                                 | Аналого-цифровой преобразователь потенциала pH/ОВП-электрода: 0–4095 (12 бит)                                                                                                                                                                                                                         |
| **Смещение (offset)**                         | Сдвиг нулевой точки pH-электрода (мВ), идеальное значение 0 при 25°C                                                                                                                                                                                                                                  |
| **Наклон (slope)**                            | Чувствительность pH-электрода (мВ/pH), идеальное значение –59.16 мВ/pH при 25°C                                                                                                                                                                                                                       |
